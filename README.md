# Madhu Portfolio

React + Vite portfolio site.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

1. Push this repo to GitHub on the `main` branch.
2. In GitHub repo settings, go to `Pages`.
3. Set `Source` to `GitHub Actions`.
4. Push to `main` again (or run the workflow manually from Actions).
5. Your site URL will appear in the workflow output.
